Step 1 :Open the Project Code

Here, My Project name is RecipeHub which is a django application. We will Create a docker image for the project directory. So Here Just open the Project Direcotry.

Step 2: Create Dockerfile

Create Dockerfile for both project.

Step 3: Write Dockerfile

Code the Docker Files:
python Environment: Uses the python 3 image.
Working Directory: Sets the working directory inside the container.
Dependencies Installation: Copies requirements.txt and install dependencies.


Application Start: Starts the application using python manage.py runserver.

Step 4: Create dockerhub repositroy

Go to hub.docker.com and create repository for the docker file.

Step 5: Build the Images

Open the terminal at each loaction of dockerfile and build the dockerimage using docker build command "docker build -t {dockerhub_username}\{repositoryname}:{tagname}" Verify image creation using docker desktop.

Step 6: Push the Image to Docker Hub

now push the image created to the repositroy created on dockerhub using docker push command "docker push {dockerhub_username}\{repositoryname}:{tagname}" Verify image creation in the repository of docker hub

Step 7: Run the image and create containar

Firstly delete all the images and containers created related to this project.
Now run the image uplaoded to docker hub using docekr hub command.
Also map the port to the external loaal port so that the project can run on local port. using -p command
Add environment varibles if any. using -e command.
Verify the Container created in docekr desktop

Step 8: Run the project Locally

As port mapping is already done while running the project, we can direcly open the port on localhost to run the servers

Dockerization of a Three-tier Application is Completed
